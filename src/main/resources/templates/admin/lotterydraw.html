Lotto Ziehung



Hier können die Lottozahlen eingetragen werden. Alle Daten müssen unter notarieller Aufsicht eingetragen werden.


Die Lottozahlen werden immer für das nächste Lottospiel gesetzt. Wenn man also 2mal hintereinander einträgt , so trägt man für die beiden kommenden Sonntage ein.
